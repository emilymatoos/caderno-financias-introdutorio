# caderno-financias-introdutorio
Este repositório reúne materiais, resumos e prompts sobre **finanças pessoais** e conceitos básicos de economia.  
O objetivo é criar um **miniguia de estudo** que apoie revisões futuras e incentive o uso da IA como ferramenta de aprendizagem ativa.

---

## 🎯 Objetivos de estudo
- Entender os principais conceitos de educação financeira.
- Diferenciar ativos e passivos.
- Aprender a montar um orçamento simples e eficaz.
- Explorar estratégias básicas de poupança e investimentos iniciais.
